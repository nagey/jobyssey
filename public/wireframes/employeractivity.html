Employer Activity

Series of tables summarizing activity in each of the following areas:

  • New Best Match Jobseekers (80% match or better for a job) [table shows code name, job name, percentage match]
  • New Applicants [table shows code name, job name, percentage match]
  • Applications needing action (table of any active applications awaiting action from employer)[shows code name, job name, stage]
  • Waiting applications (table of any active applications not awaiting action from employer) [shows code name, job name, stage]

For all tables, the code name links to the users page, the job title links to the job activity page.

NOTE: The activity manager also needs to email employers about new applications or actions, and to then give them buttons asking for action (for example, when viewing a user profile, buttons to invite or decline).
