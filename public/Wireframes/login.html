Login

Clicking on login Ajaxes (yes, I'm using it as a verb) down a second navbar with space to enter the email address and password, and a forgot password button.

Jobseekers: To the Activity section if there is new activity, to User Search Results if not.

Employers: To the Activity section if there is new activity, if not any new candidates added since they last logged on that match a job, or if not that to the jobs page.

Admins: If we could have a Google Analytics-style dashboard with site statistics and recent activity, that would be awesome!
